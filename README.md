iz.prettydate
=============

a jQuery pretty date plugin.

Usage:
------
    <script>
    // Initialize with locales, path and "now"
    $.izprettydate('loadLocale','sv','','2012-11-20T15:13+01:00');
    
    $(function(){
        $(".pretty").izprettydate();
    });
    </script>
    
    I did this <span class="pretty">2012-11-20T15:01+01:00</span>
